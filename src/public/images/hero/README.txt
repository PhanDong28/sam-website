Thư mục: public/images/hero/

Đặt file ảnh vào đây với tên chính xác:
  hero-bg.jpg

Gợi ý:
- Ảnh núi rừng / sương mù / vùng trồng sâm Ngọc Linh, tông tối để chữ vàng nổi bật.
- Kích thước tối thiểu: 1920 x 1080px (ảnh nền full-width, càng lớn càng nét trên màn hình to).
- Định dạng: .jpg hoặc .webp đều được — chỉ cần đổi lại đúng phần mở rộng trong
  src/app/page.tsx (thẻ <Image src="/images/hero/hero-bg.jpg" ...>) nếu bạn dùng .webp.
- Ảnh sẽ tự động bị phủ lớp gradient tối để chữ tiêu đề dễ đọc, nên không cần chỉnh sáng quá.

Sau khi up ảnh đúng tên, không cần sửa gì thêm trong code — trang chủ sẽ tự hiển thị.
